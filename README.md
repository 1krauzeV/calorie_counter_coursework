# calorie_counter_coursework

Учебный проект: приложение для подсчёта калорий с графическим интерфейсом и использованием базы данных.

## Планируемый функционал

- добавление продуктов;
- сохранение данных в базу;
- просмотр записей за день;
- подсчёт общей калорийности.

## Используемые технологии

- Python
- Flask
- HTML
- CSS
- SQLite
